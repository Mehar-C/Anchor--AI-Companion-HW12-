# Anchor - Anxiety-Aware AI Companion

Anchor is a mobile-first, anxiety-aware companion that connects a user's body to an AI support system in real time.

## Features

- **Apple Watch Integration**: Simulated heart rate/stress detection
- **Real-time Stress Monitoring**: Presage-powered front camera analysis
- **Adaptive AI Conversations**: Google Gemini adapts tone and strategy based on stress levels
- **Voice Support**: ElevenLabs TTS/STT for hands-free interaction
- **Personalized Coping Strategies**: DigitalOcean Gradient AI learns what works best
- **CalmToken Rewards**: Solana-based rewards for successful de-escalation

## Architecture

- **iOS App**: Main companion interface
- **watchOS App**: Heart rate monitoring and alerts
- **Shared Models**: Common data structures
- **Services**: API integrations and business logic

## Setup

1. Install dependencies via Swift Package Manager
2. Configure API keys in `Config.swift`
3. Build and run on iOS device/watchOS simulator

## API Keys Required

- Presage API key
- Google Gemini API key
- ElevenLabs API key
- DigitalOcean Gradient AI credentials
- Solana wallet/network configuration
